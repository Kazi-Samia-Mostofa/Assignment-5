Question-2:What is the difference between innerHTML,innerText and textContent?

Answer-2:Difference between innerHTML,innerText,textContent - TextContent is all text contained by an element and all its children that are for formatting purposes only. InnerText returns all text contained by an element and all its child elements. InnerHtml returns all text, including html tags, that is contained by an element.

Question-4:What is the event bubbling in the DOM?

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the parents of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object
